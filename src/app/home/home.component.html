Angular 6 + Bootstrap 4

This is a bare minimum skeleton app for starting your next Angular application.

It covers most of the key features of Angular that you use in a typical Angular App. So instead of searching for say "how to pass data from Parent to Child component" you can simply refer to the user-list and user-form component.

Most of the things that you implement in any CRUD application are covered. Click the button below to see it in action.

Installation

Just clone the repository and run npm install.

git clone https://github.com/perials/angular-quickstart-skeleton-app.git
cd angular-quickstart-skeleton-app
npm install
ng serve

Features covered

Below key features of Angular are covered in this application

  • Routing
  • Components and Services
  • Template driven forms
  • Parent Child component
  • Passing data from Parent to Child using @Input
  • Passing data from Child to Parent using @Output and Event Emitters
  • Consuming REST APIs using http
  • Handling http errors
  • Integrating Bootstrap 4 in Angular
